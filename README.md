# Turtle Crossing Game 🐢🚗
A recreation of the classic Frogger-style crossing game built with **Python** and the **Turtle** graphics module.

The objective is simple: help the turtle cross the road while avoiding incoming cars. Each successful crossing increases the game's difficulty by making the traffic move faster.

## Features

- Object-Oriented Programming (OOP)

- Random car generation

- Collision detection

- Increasing difficulty after each level

- Level counter

- Game Over screen

- Clean separation of responsibilities across classes

## Controls

- **↑ Up Arrow** — Move the turtle forward

## Project Structure

- `main.py` — Main game loop and game logic

- `player.py` — Player movement and finish-line detection

- `car_manager.py` — Car creation, movement, collision detection, and speed management

- `scoreboard.py` — Level display and game over message

## Concepts Practiced

- Classes and objects

- Composition and inheriting

- Lists for storing multiple objects and tracking their state

- Game loops

- Collision detection using `distance()`

- Random object generation

- State management

- Single Responsibility Principle

- Code organization and reusable methods
